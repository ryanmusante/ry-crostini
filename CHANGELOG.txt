CHANGELOG
=========

ry-crostini.sh -- Crostini post-install bootstrap
Lenovo IdeaPad Duet 5 Chromebook (82QS0001US / 13Q7C6)

7.7.1  2026-03-30

  Changes:
  - trim all multiline comments to single line (6 blocks)
  - trim all verbose comments (>120 chars) to concise form (40 edits)
  - no inline comments existed; no moves needed

7.7.0  2026-03-30

  Breaking:
  - remove --minimal flag: gnome-disk-utility, libavcodec-extra, and
    qemu-user are now always installed; MINIMAL variable, argument
    parsing, conditional guards, and step 11 verification guards all
    removed

7.6.4  2026-03-30

  Changes:
  - remove qt5ct from install: conflicts with QT_QPA_PLATFORMTHEME=gtk3
    set in qt.conf; qt5-gtk-platformtheme (the actual plugin) retained
  - extract _pw_gaming_content() function for PipeWire gaming config
    heredoc, eliminating 26-line duplication between fresh-write and
    upgrade-path (matches existing _gpu_conf_content pattern)
  - update --minimal description in usage() to list all three skipped
    packages (gnome-disk-utility, libavcodec-extra, qemu-user),
    matching README
  - clarify README Trixie Upgrade section: cros-guest-tools stays held
    permanently (cros-im unavailable on Trixie), not just during
    upgrade like other cros-* packages

7.6.3  2026-03-30

  Changes:
  - remove systemd v257.* APT version pin
    (/etc/apt/preferences.d/pin-systemd): systemd upgrades are no
    longer blocked; pin was originally added to prevent v258 cgroup v1
    incompatibility but is no longer desired
  - add version number to run-x86 --version and gog-extract --version
    output; version is baked in at generation time via @@VERSION@@
    placeholder sed replacement (single-quoted heredoc prevents direct
    shell expansion)

  Fixes: local-outside-function crashes, find|wc-l newline safety,
  gog-extract makeself validation, non-atomic config upgrade paths,
  nullglob guard for sources.list.d glob.
